### 图表布局（2个子图）

```
┌─────────────────────────────────────────────────┐
│                 JLP 中性套利 - 7天               │
│                 账户名 | 2024-12-04              │
├─────────────────────────────────────────────────┤

│  涨跌幅%                              净值($)   │

│  +2%|                    ~~~~           |$15.2k │

│     └──────────────────────────────→            │
│         12/01  12/02  12/03  12/04              │
│                                                 │
│   ── 净值曲线   ── 涨跌幅                        │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  回撤%                                          │
│    0%|══════════════════════╗                   │

└─────────────────────────────────────────────────┘
```

### 上图：净值曲线 + 涨跌幅（双Y轴）

| 元素 | 位置 | 说明 |
|------|------|------|
| 净值曲线 | 右Y轴 | 蓝色实线，显示美元金额 |
| 涨跌幅曲线 | 左Y轴 | 绿色虚线，显示百分比 |
| 基准线 | 左Y轴 0% | 水平虚线 |

**涨跌幅计算：**
```
涨跌幅 = (当前净值 - 首日净值) / 首日净值 × 100%
```

### 下图：回撤曲线

| 元素 | 说明 |
|------|------|
| 回撤曲线 | 红色填充区域，显示从高点的回撤百分比 |
| 最大回撤标注 | 标记最大回撤点和数值 |

**回撤计算：**
```
历史最高净值 = max(净值[:当前])
当前回撤 = (当前净值 - 历史最高净值) / 历史最高净值 × 100%
最大回撤 = min(所有回撤)
```

### 底部统计

```
收益: +$148.28 (+0.98%)  |  最高: $15,200  |  最低: $14,850  |  最大回撤: -2.5%
```
