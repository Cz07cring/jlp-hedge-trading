# 净值报告优化方案

## 优化目标

将报告从通用的"AsterDex 净值报告"改为更符合套利策略的"JLP 中性套利报告"。

---

## 优化内容

### 1. 标题修改
- **原**: 📊 AsterDex 净值报告
- **新**: 📊 JLP 中性套利报告

### 2. 新增指标

#### 2.1 JLP APR（JLP 年化收益率）
- **来源**: Hedge API 返回的 `jlp_stats.apy` 字段
- **说明**: JLP 池子本身的年化收益（来自交易手续费分成）
- **显示位置**: 账户概览区域

#### 2.2 资金费率累计收入
- **计算方式**:
  - 方案A: 从历史记录累加（需要存储每次资金费率结算）
  - 方案B: 估算 = 平均费率 × 持仓价值 × 时间（不够准确）
  - 方案C: 从交易所 API 获取（如果有这个接口）
- **说明**: 做空赚取的资金费率收入是套利核心收益之一
- **问题**: AsterDex 是否提供资金费率收入历史查询接口？
